# Rishi Dev Lab — Premium Developer Portfolio

This is a premium, recruiter-friendly, 3D-inspired portfolio website for **Rishi Shekhar**, built with plain HTML, CSS and JavaScript so it is easy to edit and deploy.

## What is included

- Animated developer hero section
- 3D-style floating code lab
- Interactive project filters
- Real GitHub repository links
- Skills universe section
- Education timeline
- Contact section with email, GitHub and LinkedIn
- Dark/light theme toggle
- Command menu with `Ctrl + K` / `Cmd + K`
- Downloadable CV text file
- Responsive mobile layout

## How to edit

Open these files in VS Code:

- `index.html` — edit text, sections, links and projects
- `styles.css` — edit design, colors, spacing and layout
- `script.js` — edit animations, filters and interactions
- `assets/Rishi_Shekhar_CV.txt` — replace with your real CV or add a PDF

## How to replace CV with PDF

1. Put your CV PDF inside the `assets` folder.
2. Rename it to `Rishi_Shekhar_CV.pdf`.
3. In `index.html`, replace:

```html
href="assets/Rishi_Shekhar_CV.txt"
```

with:

```html
href="assets/Rishi_Shekhar_CV.pdf"
```

## How to run locally

Just double-click `index.html`.

For a better development experience, open the folder in VS Code and use the **Live Server** extension.

## How to deploy

### Option 1: GitHub Pages

1. Create a GitHub repository, for example `portfolio`.
2. Upload all files.
3. Go to **Settings → Pages**.
4. Choose branch `main` and folder `/root`.
5. Save.

### Option 2: Vercel

1. Go to Vercel.
2. Import your GitHub portfolio repo.
3. Deploy as a static site.
